# Lab6 Chat IA
Este es el repositorio que contiene el codigo fuente de mi Laboratorio 6 en el cual implemento website sencillo el cual se conecta a un LLM a traves de un API de Gemini para generar texto y que le podamos hacer preguntas.

Al iniciar este proyecto queria utilizar un kit de desarrollo web que ya habiamos utilizado en clase, es por eso que escogí Vite. Los primeros commits del repositorio fueron utilizados para quitar boilerplate code que no me gustaba ya que queria que la pagina se viera la mas minimalista posible. La siguiente decisión que tome fue que LLM utilizar a travez del uso de sus developer API tools. Debido a que ya habia usado Gemini para un hackathon antes, decidí optar por esta opcion ya que ya tienia las API keys guardadas y sabia como utilizar los endpoints. El primer error con el que me enfrenté en el desarrollo de este proyecto, es que inicialmente habia creado una sesion de Gemini que servia para un prompt nada mas, lo cual no permitia tener conversaciones con el llm ya que en cada prompt la memoria se resetaba. Para ello cambie a la modalidad de chat, en la cual al iuniciar un chat con Gemini, todo lo de la sesion peude ser accedido por el modelo lo cual lo vuelve mucho mas util. Finalmente quise agregar un front end que se pareciera a una aplicación de mensajeria para que se viera facil de utilizar.
